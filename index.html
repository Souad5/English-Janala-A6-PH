Frequently Asked Questions

The difference between var, let, and const
var: Function-scoped, redeclarable, reassignable.
let: Block-scoped, reassignable, not redeclarable.
const: Block-scoped, neither redeclarable nor reassignable.
What is the difference between map(), forEach(), and filter() in JavaScript?
map() returns a new array after transforming each element.
forEach() iterates over elements without returning a new array.
filter() returns a new array containing only elements that meet a condition.
What are arrow functions, and how are they different from regular functions?
Arrow functions (=>) have a shorter syntax and do not have their own this, arguments, or super.
Regular functions have their own this, which depends on how they are called.
How do JavaScript Promises work?
A Promise represents an asynchronous operation with three states: pending, fulfilled, or rejected.
.then() handles success, .catch() handles errors, and .finally() executes regardless of success or failure.
What are closures in JavaScript?
  1. A closure is a function that remembers variables from its outer scope even after the outer function has finished execution.
  2. It allows data encapsulation and maintains state across function calls.
